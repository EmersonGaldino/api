# API-Funcional

Projeto desenvolvido com intuto de aplicar alguns patterns como:
*  DDD
*  Injection dependency
*  Design patterns
*  JWT
*  Dapper
*  AutoMapper
*  DotNet Core

## Instalação

Ao abrir o projeto faça um Restore Nuget Package.

## Utilização

Poderá ser utilizado como ferramenta de estudo!
